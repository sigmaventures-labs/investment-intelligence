# Scenario & Sensitivity Analysis — Public Equities

## Key Variables (Typical High-Sensitivity)

1. **Revenue growth** (volume × price, or organic vs. acquired)
2. **Operating/EBITDA margin** trajectory
3. **Valuation multiple** (P/E, EV/EBITDA — asset's primary metric)
4. **Capital allocation** (buybacks, M&A, capex)
5. **1-2 idiosyncratic** (regulatory, market share, product cycle)

## Return Metric

**Stock price return (%)** from current price to target price, annualized
if horizon >12 months. Present as upside/downside percentage.

## Decision Rules

- Expected return > cost of equity (~8-12%) → Interesting
- Risk/reward > 2:1 → Attractive asymmetry
- Risk/reward > 3:1 + positive EV → Strong conviction
- Bear case >30% drawdown → position sizing must reflect

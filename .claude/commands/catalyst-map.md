Build a catalyst calendar using the catalyst-mapping skill.

Target to map: $ARGUMENTS

Use the public equities overlay only. Load the public equities catalyst taxonomy from references.

Follow the SKILL.md framework:
1. Systematically check all categories in the equities catalyst taxonomy
2. For each catalyst: event, date, certainty, category, direction, magnitude, probability, thesis relevance
3. Asymmetry assessment: what's priced in, precedent, positioning → Positive/Negative/Symmetric
4. Chronological calendar view
5. Clusters, gaps, and sequencing dependencies
6. Action recommendations: pre-position, reduce/hedge, monitor, research priorities

Focus on asymmetry — expected outcome ≠ expected price/value reaction.

Note: Real estate and private equity overlays are in development and will be available in a future release.

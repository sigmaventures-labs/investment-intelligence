# investmentintelligence.ai

Landing page and product site for [Investment Intelligence](https://github.com/sigmaventures-labs/investment-intelligence).

## Stack

- Next.js 16 (App Router)
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

## Deployment

Push to `main` — Vercel deploys automatically.

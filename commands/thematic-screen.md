# Thematic Screening

Run a thematic screening analysis on:

$ARGUMENTS

Refer to @.cursor/skills/thematic-screening/SKILL.md for the core framework, then load the public equities overlay: @.cursor/skills/thematic-screening/references/public-equities.md

Follow the full framework:
1. Decompose into first, second, and third-order causal chains
2. Build asset map using the asset-class-appropriate tier structure and exposure metrics
3. Pressure-test: priced-in, counter-thesis, exposure purity, investability
4. Top 5 Deep Dive Candidates with rationale
5. Cross-asset implications if multi-asset
6. Data gaps and next steps

Prioritize non-obvious entries — the value is in the tail.

Note: Real estate and private equity overlays are in development and will be available in a future release.

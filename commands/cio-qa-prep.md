# CIO/PM Q&A Prep

Prepare an IC prep document for:

$ARGUMENTS

Refer to @.cursor/skills/cio-qa-prep/SKILL.md for the core framework, then load the public equities question bank: @.cursor/skills/cio-qa-prep/references/public-equities.md

Follow the full framework:
1. 15-25 questions from four archetypes (PM, CIO, Risk Manager, Skeptic) using asset-class-specific categories
2. At least 3 "Killer" questions
3. For each: best answer (2-3 sentences), supporting evidence, "if pressed further", red line
4. Vulnerability assessment with specific pre-meeting actions and opening framing

Concise, confident answers — not defensive essays.
